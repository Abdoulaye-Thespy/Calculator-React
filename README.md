## Setup

-To use this app locally, 

Clone this repo:

- Clone with HTTPS:
```
https://github.com/Abdoulaye-Thespy/Calculator-React.git
```
- Clone with SHH:
```
https://github.com/Abdoulaye-Thespy/Calculator-React.git
```
- run 
``` 
npm install
```
 in the directory that you download the source.

- run 
```
npm start
```
